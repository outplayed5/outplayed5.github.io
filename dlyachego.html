Для чего предназначен ИЭТР

Данное интерактивное техническое руководство предназначено для преподавателей онлайн-школы Kodland.

Оно включает в себя:

➣ общую информацию об онлайн-школе
➣ описание процесса обучения
➣ общие регламенты работы
➣ информацию о системе треков внутри школы
➣ возможные проблемы и способы их решения

Для новых сотрудников данное руководство будет неким методическим пособием, по которому будет происходить карьерное обучение.
