Руководство пользователя ИЭТР

➣ Главное меню ИЭТР состоит из 7 разделов. Для того, чтобы открыть раздел, необходимо щелкнуть по нему.

➣ Разделы 1,2,3,4 содержат справочную информацию в виде статического текста.

➣ Раздел 5 содержит информацию в виде текста и фотографий.

➣ Раздел 6 предлагает список регламентных работ, выполнение которых обязательно нужно проверить перед проведением каждого занятия.

➣ Раздел 7 выводит список возможных проблем, для решения которых система проводит интерактивный диалог с пользователем. В зависимости от полученных ответов,
система вывод рекомендации по устранению проблем и неисправностей.
